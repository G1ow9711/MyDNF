# 普通攻击三段逐帧表现设计

## 现状

战斗内核已有三段普通攻击：伤害、出手帧、动作锁、位移、击退、命中特效和第三段浮空均不同。新图集层只读取通用 `light` 和短位移进度，导致三次 X 视觉重复，并在命中后过早回到首帧。

## 动作映射

- 图集使用完整 `normalAttackStartedAtMs` 到 `normalAttackUntilMs` 计算动作进度，不再只使用命中前位移进度。
- 第一段：8 预备、9 直拳命中、8 收势，窄横向火弧。
- 第二段：9 转体、10 交叉命中、9 收势，反向交叉弧。
- 第三段：10 下沉、11 上挑命中、10 收势，竖向上升火弧。
- 每段暴露 `windup / impact / recovery`，命中阶段由权威攻击时间计算，不使用独立计时器。

## 怪物反应

- 第一段命中使用受击帧 12。
- 第二段命中使用更强侧倾帧 13。
- 第三段浮空终结使用帧 14，并继续由权威 `airborne` 状态控制位置、攻击禁止和地面阴影。

## 验收

- 真实键盘连续三次 X，依次出现段数 1/2/3 和命中提示 `ground-light-slash-1/2/3`。
- 三段人物命中帧分别为 9/10/11，怪物反应帧分别为 12/13/14。
- 第三段命中后目标进入浮空，连段计数达到 3。
- 每次按键都来自真实浏览器输入，不直接修改战斗状态。
