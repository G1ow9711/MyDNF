# 剑舞逐段音效设计

## 目标

`流光剑阵舞` 的声音必须在七个真实命中时刻播放，而不是只在按下 `Space` 时播放一次通用技能音。声音由战斗事件派生，与伤害、动作帧、怪物反应和 VFX 共用时间轴。

## 音色

- 起斩 `sword-dance-open`：拔剑金属声、短空气切割、轻冲击。
- 左舞 `sword-dance-left`：偏低的横向切割与轻碰撞。
- 右舞 `sword-dance-right`：偏高的反向切割，与左舞形成可听辨交替。
- 交叉 `sword-dance-cross`：两组错开的刃音与中低频硬直冲击。
- 终结 `sword-dance-finish`：低频主体、金属冲击、琉璃高频尾音。

七段播放顺序为 `open / left / right / left / right / cross / finish`。同一阶段命中两个目标时，以命中时间和音效 ID 去重，只播放一次。

## 验收

- 应用 Reducer 在七个命中阶段产生七条 SFX 命令，顺序严格一致。
- 每个 SFX 都有独立 WebAudio 音符计划，不得落入通用 UI 点击回退音。
- 浏览器 sink 在实际调度音符后派发只读播放事件；真实 `Space` 路线监听到七次实际播放。
- 音量继续使用主音量与 SFX 音量乘积，不绕过设置系统。
